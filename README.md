# Trần Thị Ngọc Anh · Portfolio NMCNS & AI

Portfolio học tập của **Trần Thị Ngọc Anh — MSSV 25042352**, Khoa Ngôn ngữ và Văn hóa Hàn Quốc, Trường Đại học Ngoại ngữ — ĐHQGHN.

## Thiết kế

- Concept: **Ethereal Luminescence + Glassmorphism**.
- Màu chủ đạo: Midnight Blue, Royal Blue, Sky, Porcelain và glow holographic nhẹ.
- Thumbnail Bài 1–6 dùng trang đầu của chính bài nộp, không dùng ảnh concept làm nội dung.
- Font tiêu đề là serif hệ thống hỗ trợ đầy đủ tiếng Việt; nội dung dùng sans-serif hệ thống để bảo đảm dễ đọc.
